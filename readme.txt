== Description ==
Limit Page Revisions is a WordPress plugin that allows you to control and limit the number of revisions stored for your pages. This can help keep your database clean and optimize performance by reducing the number of stored revisions.
*[About Author](https://github.com/zoynulabedin)
Key features:

- Easily set the maximum number of revisions to keep for each page.
- Improve database efficiency by limiting unnecessary revisions.
- Customizable and user-friendly settings in the WordPress admin panel.

== Installation ==
1. Upload the `limit-page-revisions` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'Limit Page Revisions' to configure the maximum number of page revisions to keep.

== Frequently Asked Questions ==
= How do I set the maximum number of page revisions to keep? =
You can configure this setting by going to 'Settings' -> 'Limit Page Revisions' in the WordPress admin panel.

= What is the minimum required WordPress version? =
This plugin requires at least WordPress 4.0.

= Does this plugin work with custom post types? =
As of version 1.0, this plugin works with pages only.
